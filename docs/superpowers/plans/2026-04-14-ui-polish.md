> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Apply Axioforce brand identity, color palette, typography, and aerospace visual language to all existing FluxDeluxe UI components.

**Architecture:** Pure styling pass — update CSS tokens, replace hardcoded colors, restyle components to match the brand spec. No logic changes, no new features.

**Tech Stack:** Tailwind CSS v4, shadcn/ui CSS tokens, Geist + Geist Mono fonts
